# 169. Majority Element

## 题目链接

- https://leetcode.com/problems/majority-element/

## 题目要求

给定 `nums`，完成 `Majority Element` 对应的 `majorityElement` 逻辑，并返回题目要求的结果。

## 示例

### 示例 1

- 输入：`nums = [3, 2, 3]`
- 输出：`3`
- 解释：本地样例用于验证你的实现是否符合题目核心规则。

## 约束条件

- `nums` 必须按原题含义处理。
- 注意空输入、重复值、边界下标和结果顺序等边界情况。

## 解题思路

- 先明确一次扫描中要维护的状态。
- 尽量把额外空间压到常数级。

## 推荐题型

- 主模式：`数组 / 贪心扫描`

## 提示

- 从左到右扫描时，先找出最关键、必须维护的状态量。
- 思考是否可以在每一步 O(1) 地更新当前最优答案。

## 复杂度目标

- 如果题目允许，优先尝试 O(n) 时间和 O(1) 额外空间。
