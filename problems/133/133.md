# 133. Clone Graph

## 题目链接

- https://leetcode.com/problems/clone-graph/

## 题目要求

给定无向连通图中某个节点，返回整张图的深拷贝。

## 示例

### 示例 1

- 输入：`adjList = [[2,4],[1,3],[2,4],[1,3]]`
- 输出：`[[2,4],[1,3],[2,4],[1,3]]`
- 解释：复制出的图结构相同，但节点对象应是新的。

## 约束条件

- 图节点值从 1 开始。
- 需要复制节点和邻接关系，不能只返回原节点。

## 推荐题型

- 主模式：`图 / BFS / DFS / 拓扑排序`

## 提示

- 写代码前先定义：图中的点和边分别表示什么。
- 判断关键状态是遍历顺序、入度，还是访问标记。

## 复杂度目标

- 图遍历类题目的常见目标是 O(V + E)。
