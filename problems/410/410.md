# 410. Split Array Largest Sum

## 题目链接

- https://leetcode.com/problems/split-array-largest-sum/

## 题目要求

给定 `nums`, `k`，完成 `Split Array Largest Sum` 对应的 `splitArray` 逻辑，并返回题目要求的结果。

## 示例

### 示例 1

- 输入：`nums = [7, 2, 5, 10, 8], k = 2`
- 输出：`18`
