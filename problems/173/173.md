# 173. Binary Search Tree Iterator

## 题目链接

- https://leetcode.com/problems/binary-search-tree-iterator/

## 题目要求

设计 `Binary Search Tree Iterator` 所要求的数据结构，并实现题目列出的公开操作。

## 示例

### 示例 1

- 输入：`operations = ['BSTIterator', 'next', 'next', 'hasNext', 'next', 'hasNext', 'next', 'hasNext', 'next', 'hasNext'], inputs = [[['7', '3', '15', 'null', 'null', '9', '20']], [], [], [], [], [], [], [], [], []]`
- 输出：`['null', '3', '7', 'true', '9', 'true', '15', 'true', '20', 'false']`
- 解释：本地样例用于验证你的实现是否符合题目核心规则。

## 约束条件

- 所有操作都作用在同一个对象实例上。
- 注意空输入、重复值、边界下标和结果顺序等边界情况。

## 解题思路

- 先定义递归函数对一个子树返回什么。
- 需要全局答案时，区分返回值和答案更新。
- 用操作序列逐步验证内部状态是否随调用正确变化。

## 推荐题型

- 主模式：`二叉树 DFS / BFS`

## 提示

- 先想清楚当前节点是在递归前、中、后哪个阶段处理。
- 明确递归函数对“一个子树”返回的含义是什么。
