# 374. Guess Number Higher or Lower

## 题目链接

- https://leetcode.com/problems/guess-number-higher-or-lower/

## 题目要求

系统选定一个 1 到 `n` 的数字，你通过 `guess` API 判断猜大、猜小或猜中，返回这个数字。

## 示例

### 示例 1

- 输入：`n = 10, pick = 6`
- 输出：`6`
- 解释：当猜到 6 时 API 返回命中。

## 约束条件

- 数字在 `[1, n]` 范围内。
- 应利用 API 反馈缩小搜索区间。

## 推荐题型

- 主模式：`二分查找`

## 提示

- 先判断你是在找精确值、边界位置，还是答案空间。
- 写代码前，先用一句话说清楚循环不变量是什么。

## 复杂度目标

- 优先尝试 O(log n) 时间复杂度。
