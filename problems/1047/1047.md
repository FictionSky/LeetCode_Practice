# 1047. Remove All Adjacent Duplicates In String

## 题目链接

- https://leetcode.com/problems/remove-all-adjacent-duplicates-in-string/

## 题目要求

给定字符串 `s`，不断删除相邻且相同的两个字符，直到不能再删除，返回最终字符串。

## 示例

### 示例 1

- 输入：`s = "abbaca"`
- 输出：`"ca"`
- 解释：先删除 `bb` 得到 `aaca`，再删除 `aa` 得到 `ca`。

## 约束条件

- 每次只能删除相邻且相同的一对字符。
- 删除后新的相邻字符也可能继续被删除。

## 推荐题型

- 主模式：`栈 / 队列`

## 提示

- 先判断题目需要的是后进先出还是先进先出。
- 如果是单调结构，先定义清楚它维护的是哪种顺序。

## 复杂度目标

- 如果题目允许，优先考虑把单次操作做到均摊 O(1)。
