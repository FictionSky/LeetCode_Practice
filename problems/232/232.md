# 232. Implement Queue using Stacks

## 题目链接

- https://leetcode.com/problems/implement-queue-using-stacks/

## 题目要求

用两个栈实现先进先出的队列，支持 `push`、`pop`、`peek` 和 `empty`。

## 示例

### 示例 1

- 输入：`operations = ["MyQueue","push","push","peek","pop","empty"]`
- 输出：`[null,null,null,1,1,false]`
- 解释：先入队的 `1` 会最先被查看和弹出。

## 约束条件

- 只能使用栈的标准操作来模拟队列。
- `peek` 返回队首但不移除。

## 推荐题型

- 主模式：`栈 / 队列`

## 提示

- 先判断题目需要的是后进先出还是先进先出。
- 如果是单调结构，先定义清楚它维护的是哪种顺序。

## 复杂度目标

- 如果题目允许，优先考虑把单次操作做到均摊 O(1)。
