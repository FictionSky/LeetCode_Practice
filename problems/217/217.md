# 217. Contains Duplicate

## 题目链接

- https://leetcode.com/problems/contains-duplicate/

## 题目要求

给定整数数组 `nums`，判断是否存在两个不同位置 `i` 和 `j`，使得 `nums[i] == nums[j]`。

## 示例

### 示例 1

- 输入：`nums = [1, 2, 3, 1]`
- 输出：`true`
- 解释：数字 `1` 出现了两次。

## 约束条件

- 数组可能为空或很长。
- 只需要判断是否存在重复元素，不需要返回重复位置。

## 推荐题型

- 主模式：`哈希表 / 哈希集合`

## 提示

- 先想清楚：在一次扫描过程中，哪些信息需要被及时记录下来。
- 考虑是否可以用额外空间换取更快的查找速度。

## 复杂度目标

- 如果查找可以做到常数时间，优先尝试 O(n) 时间复杂度。
