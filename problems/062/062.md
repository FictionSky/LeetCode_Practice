# 62. Unique Paths

## 题目链接

- https://leetcode.com/problems/unique-paths/

## 题目要求

机器人从 `m x n` 网格左上角出发，每次只能向右或向下移动，求到达右下角的不同路径数。

## 示例

### 示例 1

- 输入：`m = 3, n = 7`
- 输出：`28`
- 解释：从左上到右下共有 28 条不同路径。

## 约束条件

- 只能向右或向下移动。
- 网格尺寸为正整数。
