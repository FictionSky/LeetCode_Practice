# 55. Jump Game

## 题目链接

- https://leetcode.com/problems/jump-game/

## 题目要求

给定数组 `nums`，每个位置的值表示最多能向前跳几步，判断能否从下标 0 到达最后一个下标。

## 示例

### 示例 1

- 输入：`nums = [2,3,1,1,4]`
- 输出：`true`
- 解释：可以从下标 0 跳到 1，再跳到最后。

## 约束条件

- 每一步最多跳 `nums[i]` 格。
- 只需要判断能否到达终点。

## 推荐题型

- 主模式：`区间 / 贪心`

## 提示

- 如果区间之间的相对顺序重要，先考虑排序。
- 要明确写出贪心选择是什么，以及它为什么成立。

## 复杂度目标

- 很多区间题会先排序，所以常见目标是 O(n log n)。
