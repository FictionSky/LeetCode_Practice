# 41. First Missing Positive

## 题目链接

- https://leetcode.com/problems/first-missing-positive/

## 题目要求

给定一个未排序整数数组，找出没有出现的最小正整数，并要求尽量使用线性时间和常数额外空间。

## 示例

### 示例 1

- 输入：`nums = [1, 2, 0]`
- 输出：`3`
- 解释：本地样例用于验证你的实现是否符合题目核心规则。

## 约束条件

- 目标时间复杂度是 O(n)。
- 尽量只使用 O(1) 额外空间。

## 解题思路

- 先明确一次扫描中要维护的状态。
- 尽量把额外空间压到常数级。

## 推荐题型

- 主模式：`数组 / 贪心扫描`

## 提示

- 从左到右扫描时，先找出最关键、必须维护的状态量。
- 思考是否可以在每一步 O(1) 地更新当前最优答案。

## 复杂度目标

- 如果题目允许，优先尝试 O(n) 时间和 O(1) 额外空间。
