# 73. Set Matrix Zeroes

## 题目链接

- https://leetcode.com/problems/set-matrix-zeroes/

## 题目要求

给定矩阵，如果某个元素为 0，则将它所在的整行和整列都置为 0，要求原地修改。

## 示例

### 示例 1

- 输入：`matrix = [[1,1,1],[1,0,1],[1,1,1]]`
- 输出：`[[1,0,1],[0,0,0],[1,0,1]]`
- 解释：中间元素为 0，所以第 2 行和第 2 列都变为 0。

## 约束条件

- 必须原地修改矩阵。
- 注意不要让新写入的 0 继续扩散影响判断。

## 推荐题型

- 主模式：`矩阵遍历 / 原地修改`

## 提示

- 先判断能不能安全地在原矩阵上直接修改。
- 注意是否有行状态或列状态需要延后使用。

## 复杂度目标

- 大多数基础矩阵题可以先按 O(rows * cols) 设计。
