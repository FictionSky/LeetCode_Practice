# 378. Kth Smallest Element in a Sorted Matrix

## 题目链接

- https://leetcode.com/problems/kth-smallest-element-in-a-sorted-matrix/

## 题目要求

给定 `matrix`, `k`，完成 `Kth Smallest Element in a Sorted Matrix` 对应的 `kthSmallest` 逻辑，并返回题目要求的结果。

## 示例

### 示例 1

- 输入：`matrix = [[1, 5, 9], [10, 11, 13], [12, 13, 15]], k = 8`
- 输出：`13`
- 解释：本地样例用于验证你的实现是否符合题目核心规则。

## 约束条件

- `matrix` 必须按原题含义处理。
- `k` 必须按原题含义处理。
- 注意空输入、重复值、边界下标和结果顺序等边界情况。

## 解题思路

- 先定义堆顶代表什么。
- 再决定哪些元素入堆、何时出堆或失效。

## 推荐题型

- 主模式：`堆 / 优先队列`

## 提示

- 先定义任意时刻堆顶元素应该代表什么。
- 想清楚什么元素要入堆，以及元素何时会失效。

## 复杂度目标

- 常见目标是 O(n log n) 或 O(k log n)，取决于题目规模。
