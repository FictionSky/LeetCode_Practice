# 33. Search in Rotated Sorted Array

## 题目链接

- https://leetcode.com/problems/search-in-rotated-sorted-array/

## 题目要求

给定一个原本升序但被旋转过的无重复数组 `nums`，返回 `target` 的下标，不存在返回 -1。

## 示例

### 示例 1

- 输入：`nums = [4,5,6,7,0,1,2], target = 0`
- 输出：`4`
- 解释：`0` 位于旋转后的下标 4。

## 约束条件

- 数组中没有重复元素。
- 目标时间复杂度是 `O(log n)`。

## 推荐题型

- 主模式：`二分查找`
