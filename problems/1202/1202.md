# 1202. Smallest String With Swaps

## 题目链接

- https://leetcode.com/problems/smallest-string-with-swaps/

## 题目要求

给定 `s`, `pairs`，完成 `Smallest String With Swaps` 对应的 `smallestStringWithSwaps` 逻辑，并返回题目要求的结果。

## 示例

### 示例 1

- 输入：`s = "dcab", pairs = [[0, 3], [1, 2]]`
- 输出：`"bacd"`
- 解释：本地样例用于验证你的实现是否符合题目核心规则。

## 约束条件

- `s` 必须按原题含义处理。
- `pairs` 必须按原题含义处理。
- 注意空输入、重复值、边界下标和结果顺序等边界情况。
