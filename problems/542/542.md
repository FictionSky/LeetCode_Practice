# 542. 01 Matrix

## 题目链接

- https://leetcode.com/problems/01-matrix/

## 题目要求

给定由 0 和 1 组成的矩阵，返回每个格子到最近 0 的距离。

## 示例

### 示例 1

- 输入：`mat = [[0,0,0],[0,1,0],[1,1,1]]`
- 输出：`[[0,0,0],[0,1,0],[1,2,1]]`
- 解释：每个 1 计算到最近 0 的曼哈顿距离。

## 约束条件

- 相邻方向为上下左右。
- 矩阵中至少有一个 0。

## 推荐题型

- 主模式：`网格图 / BFS / DFS`

## 提示

- 把每个格子看成一个节点，并先想清楚邻居如何找到。
- 特别注意边界判断和访问状态更新的时机。

## 复杂度目标

- 常见目标是 O(rows * cols)。
