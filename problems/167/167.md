# 167. Two Sum II - Input Array Is Sorted

## 题目链接

- https://leetcode.com/problems/two-sum-ii-input-array-is-sorted/

## 题目要求

给定升序数组 `numbers` 和目标值 `target`，返回两个数的 1-based 下标，使它们之和等于 `target`。

## 示例

### 示例 1

- 输入：`numbers = [2,7,11,15], target = 9`
- 输出：`[1, 2]`
- 解释：`2 + 7 = 9`，下标从 1 开始。

## 约束条件

- 数组已经按非降序排列。
- 题目保证正好存在一个答案。

## 推荐题型

- 主模式：`双指针`

## 提示

- 先定义每个指针分别表示什么，再去移动它。

## 复杂度目标

- 如果不需要排序，优先尝试 O(n)；如果需要排序，常见目标是 O(n log n)。
