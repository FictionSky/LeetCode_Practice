# 225. Implement Stack using Queues

## 题目链接

- https://leetcode.com/problems/implement-stack-using-queues/

## 题目要求

用队列实现后进先出的栈，支持 `push`、`pop`、`top` 和 `empty`。

## 示例

### 示例 1

- 输入：`operations = ["MyStack","push","push","top","pop","empty"]`
- 输出：`[null,null,null,2,2,false]`
- 解释：后入栈的 `2` 会最先被查看和弹出。

## 约束条件

- 只能使用队列的标准操作来模拟栈。
- `top` 返回栈顶但不移除。

## 推荐题型

- 主模式：`栈 / 队列`

## 提示

- 先判断题目需要的是后进先出还是先进先出。
- 如果是单调结构，先定义清楚它维护的是哪种顺序。

## 复杂度目标

- 如果题目允许，优先考虑把单次操作做到均摊 O(1)。
