# 1129. Shortest Path with Alternating Colors

## 题目链接

- https://leetcode.com/problems/shortest-path-with-alternating-colors/

## 题目要求

给定 `n`, `redEdges`, `blueEdges`，完成 `Shortest Path with Alternating Colors` 对应的 `shortestAlternatingPaths` 逻辑，并返回题目要求的结果。

## 示例

### 示例 1

- 输入：`n = 3, redEdges = [[0, 1], [1, 2]], blueEdges = []`
- 输出：`[0, 1, -1]`
- 解释：本地样例用于验证你的实现是否符合题目核心规则。

## 约束条件

- `n` 必须按原题含义处理。
- `redEdges` 必须按原题含义处理。
- `blueEdges` 必须按原题含义处理。
- 注意空输入、重复值、边界下标和结果顺序等边界情况。

## 解题思路

- 先把输入关系建成邻接表或状态图。
- 根据是否有方向、入度或层次选择 DFS/BFS/拓扑。

## 推荐题型

- 主模式：`图 / BFS / DFS / 拓扑排序`

## 提示

- 判断关键状态是遍历顺序、入度，还是访问标记。

## 复杂度目标

- 图遍历类题目的常见目标是 O(V + E)。
