# 432. All O`one Data Structure

## 题目链接

- https://leetcode.com/problems/all-o-one-data-structure/

## 题目要求

设计 `All O`one Data Structure` 所要求的数据结构，并实现题目列出的公开操作。

## 示例

### 示例 1

- 输入：`operations = ['AllOne', 'inc', 'inc', 'getMaxKey', 'getMinKey', 'inc', 'getMaxKey', 'getMinKey'], inputs = [[], ['hello'], ['hello'], [], [], ['leet'], [], []]`
