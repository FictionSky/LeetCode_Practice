# 438. Find All Anagrams in a String

## 题目链接

- https://leetcode.com/problems/find-all-anagrams-in-a-string/

## 题目要求

给定字符串 `s` 和 `p`，找出 `s` 中所有是 `p` 的异位词的子串起始下标。

## 示例

### 示例 1

- 输入：`s = "cbaebabacd", p = "abc"`
- 输出：`[0, 6]`
- 解释：子串 `cba` 和 `bac` 都是 `abc` 的异位词。

## 约束条件

- 返回所有符合条件的起始下标。
- 窗口长度应与 `p` 的长度相同。

## 推荐题型

- 主模式：`滑动窗口`

## 提示

- 写代码前先定义清楚：当前窗口表示什么。
- 明确窗口在什么条件下失效，以及应该如何收缩。

## 复杂度目标

- 常见目标是单次遍历完成，也就是 O(n)。
