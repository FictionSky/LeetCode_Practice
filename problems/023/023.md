# 23. Merge k Sorted Lists

## 题目链接

- https://leetcode.com/problems/merge-k-sorted-lists/

## 题目要求

给定 `lists`，完成 `Merge k Sorted Lists` 对应的 `mergeKLists` 逻辑，并返回题目要求的结果。

## 示例

### 示例 1

- 输入：`lists = [[1, 4, 5], [1, 3, 4], [2, 6]]`
- 输出：`[1, 1, 2, 3, 4, 4, 5, 6]`
- 解释：本地样例用于验证你的实现是否符合题目核心规则。

## 约束条件

- `lists` 必须按原题含义处理。
- 注意空输入、重复值、边界下标和结果顺序等边界情况。

## 解题思路

- 先定义堆顶代表什么。
- 再决定哪些元素入堆、何时出堆或失效。

## 推荐题型

- 主模式：`堆 / 优先队列`

## 提示

- 先定义任意时刻堆顶元素应该代表什么。
- 想清楚什么元素要入堆，以及元素何时会失效。

## 复杂度目标

- 常见目标是 O(n log n) 或 O(k log n)，取决于题目规模。
