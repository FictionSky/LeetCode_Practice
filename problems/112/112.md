# 112. Path Sum

## 题目链接

- https://leetcode.com/problems/path-sum/

## 题目要求

给定二叉树根节点和目标和 `targetSum`，判断是否存在一条根到叶子的路径，路径节点值之和等于目标和。

## 示例

### 示例 1

- 输入：`root = [5,4,8,11,null,13,4,7,2,null,null,null,1], targetSum = 22`
- 输出：`true`
- 解释：路径 `5 -> 4 -> 11 -> 2` 的和为 22。

## 约束条件

- 路径必须从根节点到叶子节点。
- 叶子节点是没有左右孩子的节点。

## 推荐题型

- 主模式：`二叉树 DFS / BFS`

## 提示

- 先想清楚当前节点是在递归前、中、后哪个阶段处理。
